# macOS



1. ## 下载

前往App Store搜索“**节点小宝远程控制**”下载macOS客户端

App Store下载地址：[https://apps.apple.com/cn/app/id1632648733](https://apps.apple.com/cn/app/id1632648733)

2. ## 启动客户端

初次启动节点小宝，将自动获取绑定码若您已登录，可点击右上角菜单“设备绑定码”查看本机绑定码
